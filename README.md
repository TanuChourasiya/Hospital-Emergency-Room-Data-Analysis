# Hospital-Emergency-Room-Data-Analysis
Hospital Emergency Room data analysis using Excel with pivot tables and interactive dashboard to track patient visits and attendance trends.

**Project Overview**
This project focuses on analyzing hospital emergency room patient data using Microsoft Excel. The goal is to understand patient visits, attendance patterns, and hospital workload through pivot tables and interactive dashboards.

**Tools Used**

-Microsoft Excel
-Pivot Tables
-Charts & Visualizations
-Dashboard Design

**Dataset Description**
The dataset contains hospital emergency room records including:

-Patient ID
-Date & Time of Visit
-Patient Status (Attended / Not Attended)
-Visit Details
-Emergency Room Activity

**Key Insights**

-Identified peak patient visit periods
-Analyzed attendance patterns
-Helped understand hospital workload distribution
-Improved visibility using dashboard visualization

**Project Outcome**

This project helps hospital management to:

-Track patient flow efficiently
-Improve emergency room planning
-Optimize staff allocation
-Make data-driven decisions
